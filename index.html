ABOUT

The basic idea behind this website is to help us evolve as Web Developers. This website is going to serve as the base line for us to team up and participate in various projects and build amazing things for the web. Working as a small web development team, we hope to improve our skills and eventually work for the non-profits worldwide. This site is almost like a web-journal for all the cool stuff that we build in the process of learning, and we hope that you enjoy and appreciate all our resultant final projects.
